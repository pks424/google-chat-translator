🌐 Google Chat 번역기

Google 번역 기반 자동 번역 확장 프로그램
📤 발신 메시지 설정
내가 보낸 메시지도 번역해서 보여주기
전송된 메시지 아래에 원문 언어로 번역 표시
📥 수신 메시지 설정
상대방 메시지 자동 번역
새 메시지를 받으면 자동으로 번역하여 표시
🔑 번역 API 설정 (선택)
키 없이도 동작합니다. 더 안정적인 사용을 원하면 입력하세요.
동작 방식
• 메시지 입력 후 Enter → 자동 번역 후 전송
• 상대방 메시지 → 수신 언어로 자동 번역 표시
• Alt+T → 전송 없이 번역 미리보기
📤
자동 번역 후 전송핵심
한국어로 입력하고 Enter를 누르면 영어·일본어·중국어 등으로 자동 번역되어 전송됩니다.
📥
수신 메시지 자동 번역
상대방의 외국어 메시지를 실시간으로 한국어로 번역하여 원문 아래에 표시합니다.
🌍
8개 언어 지원
한국어, 영어, 일본어, 중국어(간·번체), 독일어, 프랑스어, 스페인어 상호 번역 지원
🔑
무료 사용 가능
API 키 없이 바로 사용할 수 있습니다. Google Cloud API 키 입력 시 더욱 안정적으로 동작합니다.
