Privacy Policy

Last updated: March 12, 2026

GChat Auto Translator ("the Extension") is a Chrome browser extension that automatically translates messages in Google Chat. This Privacy Policy explains what data the Extension collects, how it is used, and how it is protected.

1. Data We Collect

The Extension collects and processes the following data solely for the purpose of translation:

  • Message text – Text content of Google Chat messages is sent to Google's translation services to perform translation. Message text is never stored by the Extension.
  • User preferences – Language settings (source and target language), toggle preferences (auto-translate on/off, show outgoing translation on/off), and optionally a Google Cloud Translation API key are stored locally in your browser using chrome.storage.local. This data never leaves your device unless you explicitly provide a Cloud API key for use with Google's official translation service.

2. Data We Do NOT Collect

  • We do not collect, store, or transmit any personal information to our own servers.
  • We do not track browsing history or activity beyond what is necessary to translate visible Google Chat messages.
  • We do not share any data with third parties other than Google's translation APIs (see Section 3).

3. Third-Party Services

The Extension sends message text to one of the following Google translation services:

  • Google Cloud Translation API (optional, requires user-provided API key) – Used when you supply your own Google Cloud API key in the Extension settings. Subject to Google Cloud Privacy Notice.
  • Google Translate (free endpoint) – Used as a fallback when no API key is provided. This is an unofficial endpoint; usage is subject to Google's general Privacy Policy.

Message text sent to Google's translation APIs is governed by Google's privacy policies and terms of service. We encourage you to review those policies.

4. Permissions Explained

  • storage – Saves your language preferences locally in the browser.
  • host_permissions (chat.google.com, mail.google.com) – Required to read and modify Google Chat message content for translation.
  • host_permissions (translate.googleapis.com, translation.googleapis.com) – Required to send text to Google's translation services and receive translated results.

5. Data Retention

Message text is processed in-memory only and is never stored. User preference settings are stored locally in your browser and can be cleared at any time by removing the Extension or clearing browser storage.

6. Children's Privacy

The Extension is not directed at children under the age of 13. We do not knowingly collect personal information from children.

7. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated "Last updated" date.

8. Contact

If you have questions or concerns about this Privacy Policy, please open an issue on the project's GitHub repository.
